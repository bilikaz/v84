# Spacing & Radii

Geometric scale shared across the whole product — padding, margins, gaps, and border radii.

## Spacing scale

<div style={{ display: 'grid', gap: 12, marginTop: 24 }}>
  {Object.entries(spacing).map(([name, value]) => (
    <div key={name} style={{ display: 'flex', alignItems: 'center', gap: 16, padding: 12, borderBottom: '1px solid #eaeaea' }}>
      <div style={{ width: 60, fontFamily: 'ui-monospace, SFMono-Regular, monospace', fontSize: 12, color: '#6b7280' }}>
        {name}
      </div>
      <div style={{ width: 60, fontFamily: 'ui-monospace, SFMono-Regular, monospace', fontSize: 12, color: '#9ca3af' }}>
        {value}

    </div>
  ))}
</div>

## Border radii

      <div style={{ fontFamily: 'ui-monospace, SFMono-Regular, monospace', fontSize: 11, color: '#9ca3af' }}>

      </div>

  ))}
</div>

## Usage

- Use spacing tokens via Tailwind classes (`p-md`, `gap-lg`) or inline (`style={{ padding: spacing.md }}`).
- Radii: `radii.sm` for inputs and small tags, `radii.md` for buttons, `radii.lg` for cards and modals, `radii.xl` for hero surfaces, `radii.full` for pills and avatars.
- **Don't invent new values** (`padding: 18px`). If the scale doesn't cover a case, add a new token to `tokens.ts` — that's the forcing function that keeps the scale deliberate.
