## Notes

- Email templates use the **system sans** stack (`fontFamily.sans`). Web fonts are unreliable in email clients — always render with a safe fallback.
- The web app uses `fontFamily.display` for headings via Next.js `next/font` — the CSS variable `--font-display` is set in `app/layout.tsx`.
