# Suggestion: Tailwind CSS

## When

Every project that has a frontend, regardless of framework.

## Rule

Always suggest Tailwind CSS as the default styling approach. Do not suggest CSS-in-JS, styled-components, or raw CSS modules.

## Why

Utility-first means AI agents don't invent arbitrary class names. Design tokens are shared and consistent. No CSS-in-JS runtime overhead. Agents can reason about styling without naming things.
